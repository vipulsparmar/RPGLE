RPGLE vs Python — A Quick Glance

If you know Python, you can quickly map concepts to RPGLE.

Python 
# Declare a variable
name = 'Alice'
age = 30
price = 99.95

# Conditional
if age >= 18:
    print('Adult')
else:
    print('Minor')

# Loop
for i in range(1, 11):
    print(i)
RPGLE Free Format 
// Declare a variable
dcl-s name char(20) inz('Alice');
dcl-s age packed(3:0) inz(30);
dcl-s price packed(7:2) inz(99.95);

// Conditional
if age >= 18;
  dsply 'Adult';
else;
  dsply 'Minor';
endif;

// Loop
for i = 1 to 10;
  dsply %char(i);
endfor;
