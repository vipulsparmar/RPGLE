# RPGLE Free Format Documentation

A modern, comprehensive tutorial and reference for RPGLE Full Free Format programming on IBM i.

## Overview

This project provides a structured learning path for RPGLE developers, moving from basics to advanced topics like file operations, data structures, and procedures.

## Features

- **Modern UI**: Clean, responsive design with glassmorphism and smooth animations.
- **Code Examples**: Practical examples for all major RPGLE concepts.
- **Structured Learning**: 10 chapters covering everything from "Getting Started" to "Best Practices".
- **Free Format**: 100% focused on modern Free Format syntax (V7R4+).

## Deployment

This site is hosted on GitHub Pages. You can view the live documentation [here](https://YOUR_USERNAME.github.io/RPGLE/).
