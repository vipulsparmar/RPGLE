Built-in Functions (BIFs)

BIFs are RPGLE's built-in functions — they all start with %. Think of them like Python's built-in functions (len(), str(), int()) but prefixed with a percent sign.

Quick Reference

BIF Purpose Python Equivalent
%trim Remove leading/trailing blanks str.strip()
%scan Find substring position str.find()
%subst Extract substring str[start:end]
%replace Replace part of string str.replace()
%len Get length len()
%char Convert to character str()
%dec Convert to decimal float() / Decimal()
%int Convert to integer int()
%date Convert/get date datetime.date()
%elem Number of array elements len(list)
%lookup Search array list.index()

String BIFs

%TRIM / %TRIML / %TRIMR

RPGLE
**free
clean = %trim(data);      // Remove both ends
left  = %triml(data);     // Remove leading
right = %trimr(data);     // Remove trailing
strip = %trim(data : '-'); // Remove specific char

return;

%SCAN — Find Substring

RPGLE
**free
dcl-s text varchar(50) inz('Hello World');
dcl-s pos  int(10);

pos = %scan('World':text);
dsply ('Pos: ' + %char(pos));
pos = %scan('xyz':text); // 0 (not found) pos = %scan('l':text:5); // 10 (start search at pos 5) // Python: text.find('World') → 6 (0-based)

%SUBST — Extract Substring

RPGLE
dcl-s text varchar(50) inz('Hello World');

%subst(text:1:5)    // 'Hello'  (start:length)
%subst(text:7)       // 'World'  (start to end)

// Python: text[0:5]  → 'Hello'  (start:end, 0-based)
// Note: RPGLE uses start:length, Python uses start:end

%REPLACE — Replace in String

RPGLE
dcl-s text varchar(50) inz('Hello World');

// %replace(replacement : source : start : length)
text = %replace('RPGLE':text:7:5);
// Result: 'Hello RPGLE'

%LEN — Get/Set Length

RPGLE
dcl-s text varchar(100) inz('Hello');

%len(text)     // 5 (current length for varchar)

// Truncate a varchar
%len(text) = 3;  // text is now 'Hel'

Numeric BIFs

RPGLE — Numeric Gallery
**free
val = %abs(input);        // Absolute
rem = %rem(val1 : val2);   // Remainder
div = %div(val1 : val2);   // Integer Division
sqr = %sqrt(val);         // Square Root

return;

Conversion BIFs

These are your "casting" functions — like Python's str(), int(), float():

RPGLE — Conversion Gallery
**free
s_val = %char(n_val);      // Number to String
n_val = %dec(s_val:7:2);   // String to Packed
i_val = %int(s_val);       // String to Integer
u_val = %uns(s_val);       // String to Unsigned
d_val = %date(s_date);     // String to Date
return;

Date / Time BIFs

RPGLE — Date/Time Gallery
**free
today = %date();           // Get current date
now   = %time();           // Get current time
ts    = %timestamp();      // Get current timestamp

// Arithmetic
nextMonth = today + %months(1);
daysDiff  = %diff(d1:d2:*days);
yearPart  = %subdt(today:*years);
return;

Array BIFs

RPGLE — Array BIF Gallery
**free
count = %elem(myArray);    // Get size
index = %lookup('search':myArray); // Find element
%sorta myArray;            // Sort array
return;

Other Useful BIFs

BIF Purpose Example
%found Was last file op successful? if %found(CUSTFILE);
%eof End of file reached? dow not %eof(CUSTFILE);
%error Did an error occur? if %error;
%open Is file open? if %open(CUSTFILE);
%size Size of variable in bytes bytes = %size(myVar);
%alloc Allocate memory ptr = %alloc(1024);
%realloc Resize allocated memory ptr = %realloc(ptr:2048);
%xlate Translate characters %xlate('abc':'ABC':str)
%check First char NOT in list %check('0123456789':str)
