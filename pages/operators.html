Operators & Expressions

RPGLE free format uses standard arithmetic, comparison, and logical operators very similar to Python and C. This chapter covers every operator you'll need.

Arithmetic Operators

RPGLE supports all standard math operations. Here is a gallery of common arithmetic tasks:

RPGLE
**free
dcl-s num1 int(10) inz(10);
dcl-s num2 int(10) inz(20);
dcl-s sum  int(10);

sum = num1 + num2;
dsply ('The sum is: ' + %char(sum));
return;
Operator Description RPGLE Example Python Equivalent
+ Addition result = a + b; result = a + b
- Subtraction result = a - b; result = a - b
* Multiplication result = a * b; result = a * b
/ Division result = a / b; result = a / b
** Exponentiation result = a ** 2; result = a ** 2
%rem Remainder (Modulo) result = %rem(a:b); result = a % b
RPGLE — Arithmetic Gallery
**free
total = price + tax;        // Addition
net = gross - deduct;      // Subtraction
area = length * width;      // Multiplication
avg = sum / count;          // Division
cubed = x ** 3;             // Exponentiation
rem = %rem(val:2);          // Remainder (Modulo)

// Financial patterns
interest = balance * rate;
markup = cost * 1.25;
return;

Assignment Operators

Operator Description Example
= Assign value x = 10;
+= Add and assign x += 5; // x = x + 5
-= Subtract and assign x -= 3; // x = x - 3
*= Multiply and assign x *= 2; // x = x * 2
/= Divide and assign x /= 4; // x = x / 4
**= Power and assign x **= 2; // x = x ** 2
RPGLE — Assignment Gallery
**free
val = 10;           // Simple assignment
val += 5;          // Add and assign (15)
val -= 2;          // Subtract and assign (13)
val *= 2;          // Multiply and assign (26)
val /= 13;         // Divide and assign (2)
val **= 3;         // Power and assign (8)
return;
✅ Just Like Python!

RPGLE's compound assignment operators (+=, -=, etc.) work exactly the same as Python's. This is one of the most "modern" features of RPGLE free format.

Comparison Operators

RPGLE Meaning Python
= Equal to ==
<> Not equal to !=
> Greater than >
< Less than <
>= Greater than or equal >=
<= Less than or equal <=
⚠️ Important Difference

In RPGLE, = is used for both assignment AND comparison! The compiler determines meaning from context: inside if/dow/dou it's comparison; otherwise it's assignment. Python uses = for assignment and == for comparison.

RPGLE — Comparison Gallery
**free
if val = 10;    // Equality check
if val <> 5;   // Inequality check
if score > 90; // Greater than
if temp < 0;   // Less than
if stock >= 100; // Greater or Equal
if limit <= 500; // Less or Equal
return;

Logical Operators

RPGLE Meaning Python
and Logical AND and
or Logical OR or
not Logical NOT not
RPGLE — Logical Gallery
**free
// Compound logic
if (age >= 18 and hasAccount);
  // Case 1
endif;

if (isAdmin or isOwner);
  // Case 2
endif;

if (not isLocked);
  // Case 3
endif;
return;

String Operations

String concatenation uses the + operator (same as Python):

RPGLE — String Gallery
**free
fullName = %trim(first) + ' ' + %trim(last);
msg = 'Error: ' + %char(errorCode) + ' - ' + errorMessage;
path = root + '/' + file;
return;

Operator Precedence

From highest to lowest (same intuition as math and Python):

Priority Operators
1 (Highest) () Parentheses
2 ** Exponentiation
3 not Logical NOT
4 * / Multiplication, Division
5 + - Addition, Subtraction
6 = <> > < >= <= Comparison
7 and Logical AND
8 (Lowest) or Logical OR
RPGLE — Precedence Gallery
**free
res1 = 10 + 5 * 2;          // result is 20
res2 = (10 + 5) * 2;        // result is 30
check = age > 18 and score > 50;
return;
