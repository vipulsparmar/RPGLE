Variables & Constants

In RPGLE, every variable must be explicitly declared before use. This chapter covers dcl-s for standalone variables, dcl-c for constants, initialization, naming conventions, and scope rules.

Declaring Variables with DCL-S

dcl-s (Declare Standalone) is used to declare individual variables. The syntax is:

Syntax
dcl-s variableName dataType(parameters) keywords;

1. The Fundamentals: Basics

Let's start with the simplest possible declaration. We declare a variable, give it a value, and display it.

RPGLE
**free
dcl-s message char(20);

message = 'Hello RPGLE!';
dsply message;

return;

2. Practical Declaration Examples

Here are many different ways to declare variables across different domains:

RPGLE — Snippet Gallery
**free
// User Profile
dcl-s userId      int(10);
dcl-s username    varchar(25);
dcl-s emailAddr   varchar(100);
dcl-s lastLogin   timestamp;

// Inventory Management
dcl-s skuCode     char(12);
dcl-s stockQty    int(10);
dcl-s unitPrice   packed(9:2);
dcl-s isDiscounted ind;

// Financial Transactions
dcl-s accountNum  packed(20:0);
dcl-s balance     packed(15:2);
dcl-s pendingAmt  packed(15:2);
dcl-s currency    char(3);

// Date Math Variables
dcl-s birthDate   date;
dcl-s today       date inz(*sys);
dcl-s daysDiff    int(10);

// Flags
dcl-s foundError  ind inz(*off);
dcl-s fileEnd     ind inz(*off);
dcl-s isValid     ind inz(*on);

return;
📝 Python Comparison

In Python, you just write price = 0. In RPGLE, you must declare type first: dcl-s price packed(7:2);. Think of dcl-s as Python's type hints on steroids — enforced at compile time.

Initialization with INZ

The inz keyword sets an initial value. Without it, numerics default to 0, characters to blanks, indicators to *off, and dates to the system default.

RPGLE — Initialization Gallery
**free
dcl-s user     char(30)     inz('Antigravity');
dcl-s host host">rate     packed(7:2)  inz(15.50);
dcl-s host host host host host host">isDone   ind          inz(*off);
dcl-s host host host host host host host host host host host host host host">today    date         inz(%date());
dcl-s host host host host host host host host host host host host host host host host host host">empty    varchar(100) inz('');

return;

Naming Rules & Conventions

Rule Details
Max length Up to 4096 characters (practically unlimited)
First character Must be a letter (A-Z) or underscore (_)
Subsequent Letters, digits (0-9), underscores
Case sensitivity Not case-sensitive — myVar = MYVAR = MyVar
Special characters #, @, $ are allowed but discouraged

Recommended Naming Conventions

RPGLE
// camelCase (most popular in modern RPGLE)
dcl-s customerName    char(50);
dcl-s totalAmount     packed(11:2);
dcl-s isOrderComplete ind;

// Prefix for global variables
dcl-s g_companyName   char(50);

// UPPERCASE for constants
dcl-c MAX_RETRIES     5;
dcl-c TAX_RATE        0.18;

Constants with DCL-C

Constants are values that never change during program execution. Use dcl-c to declare them. The compiler infers the type automatically.

RPGLE — Constants Gallery
**free
dcl-c MAX_ITEMS   100;
dcl-c host host host host host host">VERSION     '2.0.4';
URL         'https://ibm.com';
IS_DEBUG    *on;
PI          3.14159;

return;
Python
# Python convention (not enforced)
MAX_RECORDS = 999
TAX_RATE = 0.18
APP_NAME = 'My App'
RPGLE
// RPGLE enforced constants
dcl-c MAX_RECORDS 999;
dcl-c TAX_RATE    0.18;
dcl-c APP_NAME    'My App';

Variable Scope

RPGLE has two scopes:

  • Global — declared outside any procedure, visible everywhere in the module
  • Local — declared inside a dcl-proc, visible only within that procedure
RPGLE
**free
ctl-opt dftactgrp(*no) main(main);

// Global scope — visible everywhere
dcl-s g_appName char(30) inz('My App');

dcl-proc main;
  // Local scope — only visible in main
  dcl-s counter int(10) inz(0);
  counter += 1;
  dsply g_appName;  // OK - global variable
end-proc;

dcl-proc helper;
  // counter is NOT visible here
  dsply g_appName;  // OK - global variable
end-proc;

The LIKE Keyword

Create a variable with the same type as another variable. Similar to type() in Python but resolved at compile time.

RPGLE
**free
dcl-s basePrice  packed(9:2) inz(10.00);
dcl-s salePrice  like(basePrice);

salePrice = basePrice * 0.9;
dsply ('Sale Price: ' + %char(salePrice));

Special Values

Value Meaning Python Equivalent
*on True / Yes / '1' True
*off False / No / '0' False
*zeros All zeros 0
*blanks All spaces '' (empty)
*hival Highest possible value float('inf')
*loval Lowest possible value float('-inf')
*null Null pointer None
*inlr Last Record indicator N/A
RPGLE — Special Values Gallery
**free
isActive = *on;          // Set to True
buffer = *blanks;      // Fill with spaces
counter = *zeros;      // Set to 0
maxVal = *hival;        // 9999... or Max Date
minVal = *loval;        // 0000... or Min Date

return;
