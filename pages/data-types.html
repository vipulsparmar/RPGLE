RPGLE Data Types

Unlike Python (which is dynamically typed), RPGLE is statically typed — you must declare the data type of every variable. RPGLE provides rich numeric, character, date/time, and special types optimized for business computing.

Data Types Overview

Type Keyword Description Python Equivalent
Packed Decimal packed Compressed decimal number float / Decimal
Zoned Decimal zoned One byte per digit float / Decimal
Integer int Binary integer (3/5/10/20) int
Unsigned uns Unsigned integer int (positive)
Float float Floating point (4 or 8 bytes) float
Character char Fixed-length string str
Varchar varchar Variable-length string str
Date date Date value datetime.date
Time time Time value datetime.time
Timestamp timestamp Date + Time + microseconds datetime.datetime
Indicator ind Boolean (*on / *off) bool
Pointer pointer Memory address N/A

Packed Decimal Most Common

The most widely used numeric type in RPGLE. Packed decimal stores two digits per byte (plus a half-byte for the sign), making it space-efficient and precise for financial calculations.

RPGLE — More Examples
**free
dcl-s discount    packed(5:2) inz(12.50);
dcl-s vatRate     packed(3:1) inz(20.0);
dcl-s exchangeRt  packed(11:5) inz(1.08542);
dcl-s interestPts packed(7:4) inz(4.5000);
dcl-s weight      packed(9:3) inz(145.230);
dcl-s height      packed(5:2) inz(180.50);
dcl-s temp        packed(5:2) inz(36.65);
dcl-s distance    packed(15:2) inz(1200.45);
dcl-s rating      packed(1:0) inz(5);
dcl-s progress    packed(5:4) inz(0.9999);

return;
✅ Storage

A packed(7:2) uses only 4 bytes: (7+1)/2 = 4 bytes. Formula: ceil((digits+1)/2). This is much more compact than zoned or character representations.

Zoned Decimal

Uses one full byte per digit. Less storage-efficient but sometimes needed for compatibility with display files and external systems.

RPGLE — Zoned Gallery
**free
dcl-s deptNo    zoned(3:0) inz(120);
dcl-s storeCd   zoned(4:0) inz(9988);
dcl-s branchId  zoned(5:0) inz(44123);
dcl-s regionNo  zoned(2:0) inz(14);
dcl-s legacyID  zoned(10:0) inz(9876543210);
dcl-s oldSerial zoned(8:0) inz(12345678);

return;
📝 Packed vs Zoned

Use packed for internal calculations and storage. Use zoned when interfacing with display files, APIs, or legacy programs that expect one-byte-per-digit format.

Integer (Int)

Binary integer. Use for counters, indexes, and when you need whole numbers.

RPGLE — Integer Variations
**free
dcl-s small   int(3) inz(127);    // 1 byte
dcl-s normal  int(5) inz(32767);  // 2 bytes
dcl-s large   int(10);           // 4 bytes (Standard)
dcl-s huge    int(20);           // 8 bytes

dcl-s loopCount  int(10) inz(100);
dcl-s pageNumber int(10) inz(1);
dcl-s scoreVal   int(10) inz(500);
dcl-s userId     int(10);
dcl-s errorCode  int(10) inz(0);

return;

Unsigned Integer (Uns)

Like integer but only positive values — the full byte range is used for positive numbers.

RPGLE — Unsigned Gallery
**free
dcl-s uIndex   uns(5) inz(0);
dcl-s uCount    uns(10) inz(123456);
dcl-s uFlags    uns(5) inz(255);
dcl-s uLarge    uns(20);
dcl-s uPtrOffset uns(10);
dcl-s uKeyVal    uns(5) inz(32000);

return;

Float

IEEE floating-point. Rarely used in business — packed decimal is preferred for monetary values because float can have rounding issues.

RPGLE
dcl-s pi      float(8);  // 8-byte double precision
dcl-s ratio   float(4);  // 4-byte single precision

pi = 3.14159265358979;
⚠️ Avoid for Money

Never use float for financial calculations. Use packed or zoned instead to avoid floating-point precision errors (e.g., 0.1 + 0.2 ≠ 0.3).

Character (Char) String Type

Fixed-length character string. The most common string type. Always padded with blanks on the right to fill the declared length.

RPGLE — Fixed-Length Samples
**free
dcl-s state     char(2)   inz('NY');
dcl-s gender    char(1)   inz('M');
dcl-s zipcode   char(10)  inz('10001');
dcl-s country   char(30)  inz('USA');
dcl-s status    char(1)   inz('A');
dcl-s itemType  char(8)   inz('BOLT');
dcl-s deptCode  char(4)   inz('ACCT');
dcl-s areaCode  char(3)   inz('212');
dcl-s initials  char(3)   inz('JSP');

return;

Varchar (Variable-Length)

Stores characters with a length prefix. Gallery:

RPGLE — Varchar Gallery
**free
dcl-s v_name     varchar(50) inz('Antigravity AI');
dcl-s v_json     varchar(10000); // Large buffer
dcl-s host host host host host host">v_path     varchar(255) inz('/usr/local/bin');
dcl-s host host host host host host host host host host host host">v_comment  varchar(80) inz('Work in progress');

return;
✅ When to use Varchar

Use varchar when: building SQL statements, working with web services/JSON, string lengths vary widely, or you need Python-style string behavior without trailing blanks.

Date

Stores a calendar date. RPGLE natively supports date arithmetic — no external library needed (unlike Python's datetime module).

RPGLE — Date Gallery
**free
dcl-s d_today    date inz(%date());
dcl-s host host host host host host">d_iso      date(*iso) inz(D'2024-01-01');
dcl-s host host host host host host host host host host host host">d_usa      date(*usa) inz(D'12/31/2023');
dcl-s host host host host host host host host host host host host host host host host host host host host host host">d_eur      date(*eur) inz(D'31.12.2023');

return;

Time

Stores a time-of-day value (hours, minutes, seconds).

RPGLE — Time Gallery
**free
dcl-s t_now      time inz(%time());
dcl-s host host host host host host">t_hms      time(*hms) inz(T'23.59.59');
dcl-s host host host host host host host host host host host host">t_usa      time(*usa) inz(T'11:59 PM');

return;

Timestamp

Combines date + time + microseconds. Like Python's datetime.datetime.

RPGLE — Timestamp Gallery
**free
dcl-s ts_now     timestamp inz(%timestamp());
dcl-s host host host host host host">ts_custom  timestamp inz(Z'2024-01-01-12.00.00');

return;

Indicator (Ind) Boolean

RPGLE's boolean type. Can be *on (true) or *off (false). Equivalent to Python's True/False.

RPGLE — Indicator Gallery
**free
dcl-s found      ind inz(*off);
dcl-s host host host host host host">error      ind inz(*off);
dcl-s host host host host host host host host host host host host">eof        ind inz(*off);
dcl-s host host host host host host host host host host host host host host host host host host">isValid    ind inz(*on);
dcl-s host host host host host host host host host host host host host host host host host host host host host host host host host host">isAuth     ind;

return;

Pointer

Stores memory addresses. Used in advanced scenarios like dynamic memory allocation and API calls.

RPGLE — Pointer Gallery
**free
dcl-s p_data     pointer;
dcl-s host host host host host host">p_buffer   pointer;
dcl-s host host host host host host host host host host host host">p_callback pointer(*proc); // Procedure pointer

p_data = %addr(myVariable);
p_buffer = %alloc(1024);

return;

Python vs RPGLE Data Types

Python
# Python is dynamically typed
name = 'Alice'        # str
age = 30              # int
price = 19.99         # float
active = True         # bool
today = date.today()  # date
RPGLE
// RPGLE is statically typed
dcl-s name   char(30);
dcl-s age    int(10);
dcl-s price  packed(7:2);
dcl-s active ind;
dcl-s today  date;
