Control Structures

Control structures in RPGLE work very similarly to Python's if/elif/else, while, and for — with slightly different syntax. RPGLE also has SELECT/WHEN (like switch/case) and MONITOR (like try/except).

IF / ELSEIF / ELSE

The if statement is the most basic way to control code execution. Let's start with a simple check:

RPGLE — IF Gallery
**free
if userType = 'ADMIN';
  dsply 'Access Granted';
endif;

if balance < 0;
  dsply 'Overdrawn';
endif;

if isMember and score > 80;
  dsply 'VIP Bonus Applied';
endif;

if status = 'PENDING';
  dsply 'Waiting...';
endif;

return;

Grading Example (Comparison)

Python
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'
RPGLE
**free
dcl-s score int(10) inz(85);
dcl-s grade char(1);

if score >= 90;
  grade = 'A';
elseif score >= 80;
  grade = 'B';
else;
  grade = 'F';
endif;

dsply ('Grade: ' + grade);
✅ Key Differences from Python

• Use elseif (one word) instead of elif
• End with endif; instead of indentation
• Each clause ends with a semicolon (;)

SELECT / WHEN / OTHER

Like Python's match/case (3.10+) or a long if/elif chain. Perfect for multiple discrete checks:

RPGLE — SELECT Gallery
**free
// Payment processing gallery
select;
  when pType = 'CC';
    dsply 'Credit Card';
  when pType = 'DB';
    dsply 'Debit Card';
  when pType = 'PP';
    dsply 'PayPal';
  other;
    dsply 'Cash / Unknown';
endsl;

// Status codes gallery
select;
  when status = 200;
    dsply 'OK';
  when status = 404;
    dsply 'Not Found';
endsl;

return;
📝 Note

other is the default case (like Python's else or case _:). endsl ends the select block. No "fall-through" — only the first matching when executes.

DOW — Do While Loop

dow checks the condition before each iteration — exactly like Python's while:

RPGLE — DOW Gallery
**free
// Basic while loop
dow i <= 5;
  i += 1;
enddo;

// Reading file/list pattern
dow hasMoreItems();
  item = getNextItem();
enddo;

return;

DOU — Do Until Loop

dou checks the condition after each iteration — the loop body always executes at least once. Python has no direct equivalent (you'd use while True with a break).

RPGLE — DOU Gallery
**free
// User input pattern
dou answer = 'YES' or answer = 'NO';
  dsply 'Please enter YES or NO' '' answer;
enddo;

// Retry pattern
dou success or attempts >= 3;
  success = tryConnection();
  attempts += 1;
enddo;

return;
📝 DOW vs DOU

DOW = "Do While condition is true" (check first, like Python's while)
DOU = "Do Until condition becomes true" (check after, loop runs at least once)

FOR Loop

The for loop in RPGLE is like Python's for i in range():

Python
# Count 1 to 10
for i in range(1, 11):
    print(i)

# Step by 2
for i in range(0, 20, 2):
    print(i)

# Count down
for i in range(10, 0, -1):
    print(i)
RPGLE — FOR Gallery
**free
// Basic Range
for i = 1 to 100;
  // block
endfor;

// Custom Step (by 2)
for x = 0 to 50 by 2;
  // block
endfor;

// Downwards
for i = 10 downTo 1;
  // block
endfor;

// Array processing pattern
for idx = 1 to %elem(arr);
  total += arr(idx);
endfor;

return;

ITER & LEAVE

iter = Python's continue (skip to next iteration). leave = Python's break (exit the loop).

RPGLE — ITER & LEAVE Gallery
**free
for i = 1 to 10;
  if (status = 'SKIP');
    iter;    // Continue to next
  endif;
  
  if (errorCount > 5);
    leave;   // Break loop
  endif;
endfor;

return;

MONITOR — Error Handling (Try/Catch)

monitor is RPGLE's equivalent of Python's try/except:

Python
try:
    result = 100 / 0
except ZeroDivisionError:
    result = 0
    print('Division by zero!')
RPGLE — MONITOR Gallery
**free
monitor;
  result = 100 / divisor;
on-error 00102; // Divide by zero
  result = 0;
endmon;

monitor;
  status = processTransaction();
on-error *all;
  logError('Critical error in process');
endmon;

Common Error Codes

Code Error Type
00100 String/array errors
00102 Divide by zero
00121 Array index out of bounds
00202 Call to program failed
00211 Conversion error
*all Catch all errors
RPGLE — Multiple on-error
monitor;
  // risky code here
  value = %dec(inputStr:9:2);
  result = value / divisor;

on-error 00211;
  dsply 'Invalid number format';

on-error 00102;
  dsply 'Cannot divide by zero';

on-error;  // Catch everything else
  dsply 'Unexpected error';
endmon;
